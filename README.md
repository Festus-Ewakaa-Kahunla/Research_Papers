# Research Papers from Scratch

Breaking down and implementing foundational research papers from scratch. Each paper is studied, explained, and implemented to build a deep understanding of the core ideas behind modern machine learning and deep learning.

## Structure

Each folder contains:
- The original paper (PDF)
- A breakdown and explanation of the key ideas
- An implementation from scratch
